TikTok Sans is bundled as the default native-style caption font:

- TikTokSans-Regular.ttf
- TikTokSans-OFL.txt

You can add more licensed font files here for higher-fidelity server rendering:

- BebasNeue-Regular.ttf
- CormorantGaramond-Regular.ttf
- CormorantGaramond-Italic.ttf
- Anton.ttf
- Inter-Bold.ttf

The app works without optional fonts by falling back to system fonts in SVG rendering. For redistribution, include font licenses with any font files you add.
